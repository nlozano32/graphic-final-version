
Requirements:

By the server side:

- Apache version 1.3.27+
- PHP (PHP 4, PHP 5, PHP 7)

By the client side: 

- Mozilla Firefox (Current - 1) or Current
- Internet Explorer version 11+
- Google Chrome (Current - 1) or Current
- Opera 12.1x, (Current - 1) or Current
- iOS 6.1+
- Android 2.3, 4.0+

References: 

- https://jquery.com/browser-support/
- http://getbootstrap.com/getting-started/#support

Linux:

1. Copy this folder in /var/www/html
2. Open the browser and call the application as follows: http://IPAddressMachine/graphic/index.php

Windows:

1. Copy this folder in Apache root directory
2. Open the browser and call the application as follows: http://IPAddressMachine/graphic/index.php

Suggestions for installation in windows:

1.  Download a LAMP like XAMPP, WAMP Server and continuing with the installation described above.
